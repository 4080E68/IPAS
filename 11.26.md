# 國家標準暨技術研究院 雲計算的定義

```
按需自助服務。消費者可以單方面提供計算功能，例如
服務器時間和網絡存儲，無需人工即可自動實現
與每個服務提供商的互動。

廣泛的網絡訪問。功能可通過網絡獲得，並可通過標準訪問
促進異構瘦客戶機或胖客戶機平台使用的機制（例如，
手機，平板電腦，筆記本電腦和工作站）。

資源池。提供商的計算資源被集中起來為多個消費者提供服務
使用多租戶模型，動態地使用不同的物理和虛擬資源
根據消費者需求分配和重新分配。有位置感
獨立性在於客戶通常無法控製或不了解確切的
所提供資源的位置，但可能能夠指定更高級別的位置
抽象（例如國家/地區，州或數據中心）。資源的示例包括存儲，
處理，內存和網絡帶寬。

快速彈性。在某些情況下，可以彈性配置和釋放功能
自動，以根據需求快速向外和向內擴展。到
消費者，可用於配置的功能通常似乎是無限的，並且可以
在任何時候都可以任意數量地使用。

實測服務。雲系統通過利用來自動控制和優化資源使用
在某種抽像水平上適合於服務類型的計量能力1（例如，
存儲，處理，帶寬和活動用戶帳戶）。資源使用情況可以是
監控，控制和報告，為提供商和
使用服務的消費者。
```
