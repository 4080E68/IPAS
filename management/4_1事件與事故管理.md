# 資安事故處理流程
## 準備
```
– 組織資安事故處理小組
– 建立資安事故處理策略
– 設計資安事故處理程序
– 建立溝通管道與方式
– 蒐集所需資源
– 練習、練習、再練習
```
## 問題識別
```
偵測到問題後第一步要先進行記錄，判斷意圖、保留證據
```
## 問題調查
```
了解發生問題的來龍去脈，
```
## 問題隔離
```
「封鎖」入侵來源，以避免災害擴大
```
## 問題根除
```
解決問題發生的根本原因並加以解決
```
## 復原與經驗學習
```
一旦威脅被根除，接下來應開始將業務與服務回復至正常運作狀態
最後分析整體事件作為未來類似事件的參考依據
```
